# Hướng dẫn
### Bước 1
vào thư mục muón chứa game và mở command prompt lên và nhập:
```
git clone https://github.com/trong269/plane-game.git
```
nhấn  `Enter`
### Bước 2
mở command prompt trong thư mục game và nhập:
```
pip install -r requirements.txt
```
nhấn  `Enter`
### Bước 3
khởi chạy game
```
python game.py
```
nhấn  `Enter`
